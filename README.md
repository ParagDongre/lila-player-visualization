# LILA Player Journey Visualization Tool

This tool visualizes player movement and gameplay events from telemetry data.

Features:
- Player journey visualization
- Event markers (kills, deaths, loot etc.)
- Timeline playback
- Heatmap of player activity
- Map and match filtering
- Minimap overlay

Tech Stack:
- Python
- Streamlit
- Plotly
- Pandas

Run locally:

pip install -r requirements.txt
streamlit run app.py
